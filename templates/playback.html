Set up the Tautulli webhook

Tautulli is connected but no play events have arrived yet. Add a webhook notification agent in Tautulli to start collecting data.

  1. In Tautulli, go to Settings → Notification Agents
  2. Click Add a new notification agent and choose Webhook
  3. Set the Webhook URL to <this-server>/api/tautulli/webhook
  4. Set Method to POST
  5. Under Triggers, enable Playback Start and Watched
  6. Under Data → JSON Data, paste the JSON below
  7. Save and test — play something in Plex to verify
JSON Data payload 
{
  "event": "{action}",
  "event_at": "{unixtime}",

  "media": {
    "rating_key": "{rating_key}",
    "tmdb_id": "{themoviedb_id}",
    "media_type": "{media_type}"
  },

  "playback": {
    "transcode_decision": "{transcode_decision}",
    "video_decision": "{video_decision}",
    "audio_decision": "{audio_decision}",
    "subtitle_decision": "{subtitle_decision}",
    "quality_profile": "{quality_profile}"
  },

  "source_quality": {
    "container": "{container}",
    "video_codec": "{video_codec}",
    "video_bitrate": "{video_bitrate}",
    "video_resolution": "{video_full_resolution}",
    "video_bit_depth": "{video_bit_depth}",
    "audio_codec": "{audio_codec}",
    "audio_channels": "{audio_channels}"
  },

  "stream_quality": {
    "stream_container": "{stream_container}",

  }
}
